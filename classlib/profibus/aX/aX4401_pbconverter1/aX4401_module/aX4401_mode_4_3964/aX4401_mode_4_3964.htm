aX4401_pbc1 3964 Converter


Features

This module represents a 3964 RK512 - converter.

 

You can use follow baud rates:

  • 300 Baud
  • 600 Baud
  • 1200 Baud
  • 2400 Baud
  • 4800 Baud
  • 9600 Baud
  • 14400 Baud
  • 19200 Baud

To config the module push on  the mode name in the graphical representation.

 

Connection

The RS232 Connector has follow pin assignment :

Pin Funktion
2 RxD
3 TxD
4 RTS
5 CTS
7 Gnd
