aX4401_pbc1 Converter


Features

The Counter aX4401_pbc1 includes eight input channels, four digital output channels and a serial interface. You could add several modules to the main class:

  • Simple Counter (Mode 0)
  • Extended Counter (Mode 1)
  • Extended Dosing Counter (Mode 2)
  • RS232 - Profibus Converter (Mode 3)
  • 3964 / RK512 - Profibus Converter (Mode 4)

Implementation Notes

How to Create the Profibus Slave

The first step is to configure a Profibus Slave module. To create a Profibus Slave, take the following steps:

Step 1:
Select the Profibus Slave class and drop an instance of it into the picture window

Step 2:
With the slave object selected, enter a unique instance name and remark in the declaration field, in the configuration panel.

When you are naming I/O modules, be sure to assign meaningful names that you will be able to recognize later. For example, the Profibus Slave should at least be given an address number:
 

  • Instance name = test

Figure 22: Assign Instance name
 

Step 3:
Open the Server Data tab. Find the resource called "Master" in the data list. Click the value field with the left mouse button.

automationX automatically detects the Master and presents it in a drop down list.

Choose the master name from the list.


Figure 23: Assign the Master
 

Step 4:
Find the resource called "address" in the Server data list. Use the increment/decrement arrows to set the address number of the Slave.

For example, if the address of the Slave class were 11 you needed to set the number to 11. When you save the display, you will see this number reflected on the object`s "counter".

 Figure 24: Assign the Profibus Slave address

Specify the Counter Version

Find the resource called "ModuleConfig" in the Server Data list. Click on the resource and select your counter module from the appearing list. (Two possibilities: "pbc_1" - no comparing function, "pbc_1c" - with comparing function). Please remember that this can only be done in edit mode.
