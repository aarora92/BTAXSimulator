Troubleshooting

Install Message: "Can't load dll:MBSA85.dll"

This problem is likely occurring because the MBX driver is not installed, or has not installed correctly. The automationX Modbus driver communicates to the MBX Driver through a file called Netlib.dll.  This file is installed by the MBX Driver software and should be located in the WinNT/system32  directory on the server. If not, try reinstalling the MBX Driver.

If the problem persists, the install routine for the MBX Driver may be corrupt or not functioning properly. Contact your MBX Driver software provider for assistance.
