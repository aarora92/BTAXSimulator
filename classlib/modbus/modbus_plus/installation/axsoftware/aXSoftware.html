automationX MB+ Class Set

Directory Structure

The automationX Modbus Plus class set must reside in the class library of the project (e.g. ...Projects/MyProject/classlib/Modbus). References to certain documentation files requires that the original directory structure be maintained.

MB+ Classes

The aX Modbus Plus classes are as follows:

  • ModbusSA85: master communciation class - required for communication between the network and the client classes.
  • ReadBack: required if values are to be written from automationX to the Modbus Plus network. The ReadBack class ensures that values from the network are updated to automationX before a write operation occurs. It can also be used to monitor data integrity in the event of a network error or shutdown so that retentive data is not lost.
  • Modbus_Coil: used to send and receive coils to and from Modbus Plus network.
  • Modbus_Register: used to send and receive registers to and from Modbus Plus network.
  • Modbus_Float: used to send and receive floats to and from Modbus Plus network.
