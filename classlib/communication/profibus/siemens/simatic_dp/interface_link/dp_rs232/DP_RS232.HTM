DP_rs232: Graphic Interface

The DP_rs232 user interface includes a process picture representation and one popup menu.

The DP_rs232 process representation is shown in Figure 2.

The DP_rs232 popup menu include the error messages, shown on Figure 3.

Figure2: DP_rs232

A new error on the DP_rs232 causes the Operations Interface window,

Figure 3, to be opened on the computer screen.

Figure 3: DP_rs232 Error Message Popup

The DP_rs232 operater graphic also includes the Instance name or the Profibus DP Slave DP_rs232

and below it there is a comment describing the Profibus DP Slave unit.

The Popup shows the occuring errors below the header. It can be only closed with the "Quit" button at the bottom

of the popup to ackknowledge the errors.

DP_rs232: Implementation Notes

 

Qualities:       The DP_rs232 is characterized by the following qualities:

 

You can employ the DP rs232 for the following modes:

- Siemens Procedure 3964 R

- Controll about Signaldelaytime

- Start and Endsignal

- Constance Datalength

for more Information have a look to the file: Docu_DP_rs232.pdf
