MOBY ASM470

 

Graphical Representation:

Description of the LED top right:

Color

Description

Grey

No action.

Blue

Command will be executed.

Green

Command execution ok.

Red

Command execution failure.

The symbol in the middle right shows the presence of a MDS.

 

Popup:

A mouse click on the graphical representation opens the popup.

Function

Description

Initialisation

"Size" bytes of MDS will be initialisized with "Value".

The execuction takes place after pushing the "Init" button.

Read

Reads at "Offset" the amount of "Size" bytes from the MDS.

The execuction takes place after pushing the "Read" button.

Write

Writes at "Offset" the amount of "Size" bytes to the MDS.

The execuction takes place after pushing the "Write" button.

It is necessary, to connect a data symbol to this symbol (for example .s7_mds_E611).
