Digital Input/Output ET200B DI8/RO8 AC

This digital input/output module ET200B DI8/RO8 AC includes eight isolated inputs and eight isolated outputs.

The module summarize the individual digital inputs together to an input byte(s) and handles the output byte(s) the other way.

ET200B DI8/RO8 AC: Graphic Interface

 

The ET200B DI8/RO8 AC user interface includes a process picture representation and several popup menus. The ET200B DI8/RO8 AC process representation is shown in Figure 2. The ET200B DI8/RO8 AC popup menus include four operater interfaces. Figure 3 shows one popup for the input channels and one for the output channels. the two that are not shown are simular to the other ones.

The ET200B DI8/RO8 AC process picture has a unique name identifier ($Instance) for the ET200B DI8/RO8 AC displayed in the middle of the picture (DI16_DO16). Each channel is visualized by a small rectangle simulating a LED on the right side of the graphical representation. All channels are displayed next to the unique name identifier, the output channel at first and then the input channels . If a channel's value is forced by using the operater interface the rectangle next to the simulated LED change its color to red. The rectangles on the top left of the process picture signalled the status of the ET200B DI16. It shows if the module has the status RUN, a bus fault (BF) occurs or the diagnosis is activ (DIA).

 
Figure 2: ET200B DI8/RO8 AC Graphical Representation

 

Using the left mouse button down at one of small rectangles on ET200B DI8/RO8 AC process representation causes the Operations Interface window, Figure 3, to be opened on the computer screen.

 
Figure 3: ET200B DI8/RO8 AC Operations Interface

The ET200B DI8/RO8 AC operater graphic also includes the Instance name or the Digital Input/Output ET200B DI8/RO8 AC and below it there is a comment describing the Digital Input/Output unit. The button which hosts the Instance name and the comment can close the popup.

The operater interface is splitted in eight parts, for each channel one. The LED on the top right of each section shows the value of each channel. It is green if the real input or output is logical "1", it is red if the channel's value is logical "1" in force mode and it is grey if the value is  logical "0".
With the "Force" button the each channel can be set to force mode for its own to simulate the value, which is necessary if there is no connection to the I/O. So the PLC-programs can be tested without an I/O. The value of each channel is then simulated with the second button in the respective section. This button ("ON/OFF") is only sensitive if the "Force" button is activated.

The right text above the channel section signalled whether the module or the data is valid. This text dis appear only if the module and the data is valid.

ET200B DI8/RO8 AC: Implementation Notes

 

Qualities:       The ET200B DI 8/DO 8 is characterized by the following qualities:

- 8 inputs, isolated
- 8 outputs, isolated
- DC 24 V entry nominal voltage
- DC 24 V burden nominal voltage
- entry delay: 3 ms
