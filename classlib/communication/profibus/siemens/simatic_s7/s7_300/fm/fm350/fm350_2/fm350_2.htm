Counter FM 350-2

Features

This counter module FM 350-2 includes eight channels, eight hardware gates and eight outputs. This class can be used in a S7-300 or Profibus DP Slave-System (ET200M IM 153).

User Interface

The Counter FM 350-2 class represents an actual FM 350-2 module. The instance name will appear at the middle of the module. If the module is not connected or not valid, the instance name will be flashing red.

The eight output points are  numbered from Q0 - Q7 and the eight hardware gates are numbered form I0 - I7, each channel represented by a small rectangle. The rectangles denote the current state of the channel: grey if inactive, green if set.

The Initiation Popup: The required parameters can be adjusted here:

The below described LED's display the actual counter state:

  • STS_CMP: status comparator
  • STS_DIR: status counting direction

  • STS_DI: status digital input
  • STS_DO: status digital output
  • STS_GATE: status internal gate
The following buttons allow to switch specific functions:
  • CTRL_DQ: control the digital output
  • SET_DQ: set the digital output
  • SW_GATE: software gate open/close
  • TFB:
  • RES_STS: Reset status-signal

Implementation Notes

Data Transfer will only work correctly when the counter module FM 350-2 is connected with at least one Module or Slave.  The system will check for an existing module or slave automatically and display the name at the bottom of the graphical representation.

Link the I/O Blocks Together

Once you have placed all of the necessary I/O modules on to the display , you will need to connect them. The I/O objects in AutomationX communicate in series just like real I/O blocks, starting from the Profibus Slave.

Each I/O module has a "Connection" resource in the Server Data list. To connect your counter module FM 350-2 , start with the Profibus Slave  and type the instance name of the next module (in the sequence ) into the "Connection" field. The "Connection" field  of the last module in the sequence can be blank.

 

For a detailed description of the counter please refer to the Siemens documentation (Configuration Package CD - delivered with the counter module)
