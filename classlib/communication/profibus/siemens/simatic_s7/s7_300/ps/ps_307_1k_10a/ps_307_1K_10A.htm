ps_307_1K_10A

Features

The ps_307_1K_10A class represents a SITOP power supply in a S7-300 or a Profibus DP Slave-System (ET200M series) system.

Features | Installation



Installation

 The ps_307_1K_10A class is comprised of the following components (files):
 
 
Class Component Description File Name(s)
Class Color File Includes some class specific color definitions. ps_307_1K_10A.color
Class File Includes the process picture representation and any popup menus. ps_307_1K_10A.symbol

 
HTML Document HTML document for Online Help. ps_307_1K_10A.html

Installation: The above listed files must be copied into a common sub directory within the project's "classlib" directory. The name of the common directory can be chosen freely.
