Analog Input/Output SM334 AI4/AO2 x 8/8Bit

This analog input/output module SM 334 AI4/AO2 includes four inputs and two outputs with a resolution of 8 bit.

It can be used in a S7-300 or in a Profibus DP Slave-System ( ET200M IM153 ).

The module converts the analog process signal to a digital value for the analog inputs and for the analog outputs the other way with the selectet resolution.

SM334 AI4/AO2: Graphic Interface

 

The SM334 AI4/AO2 user interface includes a process picture representation and several popup menus. The SM334 AI4/AO2 process representation is shown in Figure 2. The SM334 AI4/AO2 popup menus include two operater interfaces, shown by Figure 3. The first one handles the analog inputs and the second the analog outputs.

The SM334 AI4/AO2 process picture has a unique name identifier ($Instance) for the SM334 AI4/AO2 displayed in the middle of the picture (SM334_4AI_2AO). Each channel is visualized by a slider and a value display. The first three input channels are displayed above the unique name identifier. The fourth input channels and the two output channels are visualized below. If a channel's value is forced by using the operater interface the rectangle above the value display change its color to red. The rectangle at the top right signalled a service fault and the black rectangle at the bottom left shows the module position.

Figure 2: SM334 AI4/AO2 Graphical Representation

 

Using the left mouse button down at one of sliders above the unique name identifier on SM334 AI4/AO2 process representation causes the Operations Interface window for the inputs, Figure 3, to be opened on the computer screen. Using the left mouse button down at a slider below the unique name identifier causes the same for the output channel Operation Interface window.

Figure 3: SM334 AI4/AO2 Operations Interface

The SM334 AI4/AO2 operater graphic also includes the Instance name or the Analog Input/Output SM334 AI4/AO2 and below it there is a comment describing the Analog Input/Output unit. The button which hosts the Instance name and the comment can close the popup.

The first operater interface for the inputs is splitted in four parts and the second operater interface for the outputs into two parts, for each channel one. The right slider shows always the real input or output value. With the button at the bottom of each channel section the mode could be changed from normal mode to force mode. Then the input or output value can be force by the left slider which isn't sensitive as long as the button shows "Norm". This means that input/output value is simulated with a variable value edited by a slider to test the plc programs without any I/O's.

The right text above the channel section signalled whether the module or the data is valid. This text dis appear only if the module and the data is valid.

SM334 AI4/AO2: Implementation Notes

 

Qualities:       The SM334 AI4/AO2 x 8/8Bit is characterized by the following qualities:

- 4 entries, 2 exits
- resolution 8 bits
- measuring range optional of 0 to 10 V or 0 to 20 mA
- output range optional of 0 to 10 V or 0 to 20 mA
- any choice between tension and current
- non-isolated to the CPU
- isolated to the load supply
