ActivBus

Features

The ActivBus class is used in an ET200M Profibus active bus system to emulate an empty slot.

Installation

 The ActivBus class is comprised of the following components (files):
 
 
Class Component Description File Name(s)
Class Color File Includes some class specific color definitions. ActivBus.color
Class File Includes the process picture representation and any popup menus. ActivBus.symbol

 
HTML Document HTML document for Online Help. ActivBus.html

Installation: The above listed files must be copied into a common sub directory within the project's "classlib" directory. The name of the common directory can be chosen freely.

User Interface

The ActivBus class represents an the active bus module of the ET200M active bus system.

Implementation Notes

To emulate an emtpy slot in a active bus system the ActivBus module must be connected in series with the next IO-module. To do this , assign the "Instance" name of th next module into the "Connection" string resource of the current module. If there is no IO-module next in the series, the "Connection" resource remains blank
