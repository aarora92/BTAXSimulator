PID Controller

The Regulatory PID Controller is used to perform standard control functions using the standard PID (Proportional Integral Derivative) control algorithm. The Regulatory PID Controller in AutomationX is designed to include a wide range of configurable options and this allows the implementation of complex control strategies.

The regulatory controller can be implemented as a single PID controller, a cascaded controller, or a feedforward controller. In addition, the proportional, the integral and the derivative component of the PID controller operate independently of each other. Each portion of the PID controller can be disabled or enabled to realize P, I, D, PI, PD or PID control.

Details of the PID algorithm are not discussed here. Most books on process control include a description of the PID algorithm. The following hyperlinks explain the PID implementation in AutomationX.
