PID Configuration

The following table lists the configuration parameters for the Regulatory PID controller. The parameter data type, the default values and a brief description are listed.

 

Item Variable Type Default Description
1 value_in REAL 0 measured value - feedback
2 setpoint_in REAL 0 setpoint
3 Dv REAL 0 disturbance value
4 control_out REAL 0 controller output
5 control_extern REAL 0 external assigned control value.
6 control_extern_enable BOOL NO enable external control value
7 control_internal REAL 0 internal control value with a range of 0-100%
8 dv_internal REAL 0 internal disturbance value with a range of 0-100%
9 setpoint_visual REAL 0 setpoint for visualization. In manual setpoint mode , this value is used as setpoint and can be changed by the operator on a popup.
10 value_visual REAL 0 measured value after smoothing
11 control_visual REAL 0 control value for visualization. In manual mode, this value is used as output value and can be changed by the operator on a popup.
12 Kp REAL 1 proportional factor (gain)
13 Ti REAL 10.0 time constant of integrational part
14 Td REAL 1.0 time constant of derivative part
15 td_d REAL 0 delay time of derivative part
16 tu_sp REAL 0 time for runup ramp- setpoint ( MIN_SCALE to MAX_SCALE)
17 td_sp REAL 0 time for rundown ramp- setpoint ( MAX_SCALE to MIN_SCALE)
18 tu_con REAL 0 time for runup ramp- control value (control_min to control_max)
19 td_con REAL 0 time for rundown ramp- control value(control_max to control_min)
20 ts REAL 0 smoothing time for measured value
21 control_min REAL 0.20 minimal control value
22 control_max REAL 100.0 maximal control value
23 integral_off REAL 100.0 if error signal exceeds this value the integral part is disabled
24 error_deadband REAL 0 deadband width for error signal
25 error_last_cycle REAL 0 error signal in last plc-cycle
26 I_last_cycle REAL 0 integral value in last plc-cycle
27 d_last_cycle REAL 0 derivative value in last plc-cycle
28 sp_last_cycle REAL 0 setpoint in last plc-cycle
29 control_last_cycle REAL 0 control value in last plc-cycle
30 setpoint_actual REAL 0 actual setpoint
31 value_actual REAL 0 actual measured value (smoothed)
32 error_actual REAL 0 actual error signal
33 p_actual REAL 0 actual proportional part
34 I_actual REAL 0 actual integral part
35 d_actual REAL 0 actual derivative part
36 auto BOOL NO enables the automatic mode
37 sp_auto BOOL NO enables the automatic mode for setpoint
38 shutdown BOOL NO set the setpoint to zero
39 freeze BOOL NO froze the control value
40 enable BOOL YES enables the controller, otherwise control value = control_min
41 p_enable BOOL YES enables the proportional part
42 I_enable BOOL YES enables the integral part
43 d_enable BOOL NO enables the derivative part
44 $$_cycle_time BOOL NO system-wide  plc cycle time
45 SPS_Constuctor Code   class code
