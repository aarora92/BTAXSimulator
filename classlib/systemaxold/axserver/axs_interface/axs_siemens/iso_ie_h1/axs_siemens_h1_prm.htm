Industrial Ethernet Status Popup

With a mouse click on the active Ethernet card slot of the Siemens panel you will get the Ethernet Status Popup.
 


In the Ethernet status popup you can see the Ethernet status string (e.g. CARD OPEN) in the field Status, the service access point in the field SR Access Point and the MAC address in the field MAC Address.
You can see also the number of active slaves in the field Active Slaves, the sum of the input data in the field Sum of Input data and the sum of the output data in the field Sum of Output data.
With the buttons Open and Close you can open and close the Siemens Ethernet card.
With the button Help you will get this online documentation of the Ethernet setting and with the button Exit you can close the popup.
 
Status field color Description
Green Communication okay
Red (flashing) Communication not okay
Light Grey No Card
