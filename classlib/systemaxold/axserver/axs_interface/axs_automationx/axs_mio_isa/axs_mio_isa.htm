Graphical User Interface

MIO ISA Panel

The MIO_ISA panel displays if there is an ISA-Card existing and denotes the card types in the 6 slots.
 


The LED display at the top of the panel indicates state of the plc.
The following table provides a summary:
 
LED Color Description
Green The plc is running.
Red The plc is not running.

The displays below are indicating the appointed MIO-modules.

Implementation Notes

The axs_mio_isa class is an integral part of the "axsystem" program. In terms of implementation it is required to have the "axsystem.tree" program installed into the running project. I.e. in the "Configuration" directory of the Project Window.
