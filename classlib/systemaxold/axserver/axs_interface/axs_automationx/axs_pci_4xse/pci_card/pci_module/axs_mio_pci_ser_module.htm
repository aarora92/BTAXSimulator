Serial Status Popup

With a mouse click on the SER_PCI panel you will get the serial status popup.

 

In the serial status popup you can see the physical mode ( e.g.: RS422 ) in the field Physical Mode, the protocol name ( e.g.: modbus ) in the field Protocol Name and the ID of the protocol in the field ID for each channel. With the button Reset Mode you can also reset each channel of your serial card. You can also see the sum of input data in the field Input Data and the sum of output data in the field Output Data of all channels together. In the field Running Counter you can see if you SER_PCI card is running.
With the button Help you will get this online documentation of the serial settings and with the button Exit you can close the popup.
 
 
Status field color Description
Green Communication okay
Red (flashing) Communication not okay
Light Grey No Card

If your serial card doesn't run, your serial running counter will stop, and the staus field color will flash red.
