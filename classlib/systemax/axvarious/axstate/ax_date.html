Features

The ax_date class displays the current system date with the parameters

  • day
  • month and
  • year
Features | Installation | User Interface



Installation

 The ax_date class is comprised of the following components (files):
 
 
Class Component Description File Name(s)
Class File Includes the process picture representation and any popup menus. ax_date.symbol

 
HTML-Document HTML document for Online Help. ax_date.htm

 

Installation: The above listed files must be copied into a common sub-directory within the project's "classlib" directory. The name of the common directory can be chosen freely.
