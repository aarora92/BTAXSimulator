Touch Screen Letter Keys

Features

With this class you can input letters, numbers and special characters in a own ledge without a keyboard
directly on the screen. That means, you are working directly on the screen with a popup which is similar built as
a keyboard. Precondition is that you are using a Touch Screen.

Graphical User Interface

The TS_LetterKeys class represents an actual keyboard on your screen. The instance name of the module will appear
 in white text on the top right of the symbol.

First you set the class in your tree and a text ledge will appear.
 

Figure 1: Graphical Representation
 

The PopUp is invoked by a click on the graphical representation.
This popup contains a picture with a keyboard with all of the letters, numbers and special characters.

Figure 2: Popup
 

Also the keyboard contains a Shift Button for second funtions, e. g. for capital letters, quotation mark, parenthesis
and another characters.

Figure 2: Popup (Shift Button active)
 
 

Further there is a AltGr Button for third functions. If this funcion is active, the keys contains special characters only.

Figure 3: Popup (AltGr Button active)
 

Note: The TS_LetterKeys class is also working with Multi Language!
