Touch Screen Real Keys

Features

With this class you can input real numbers and change the sign in a own ledge without a keyboard
directly on the screen. That means, you are working directly on the screen with a popup which is similar built as
the number block on the keyboard. Precondition is that you are using a Touch Screen.

Graphical User Interface

The TS_RealKeys class represents a number block on your screen. The instance name of the module will appear
 in white text on the top right of the symbol.

First you set the class in your tree and a ledge will appear.
 

Figure 1: Graphical Representation
 

The PopUp is invoked by a click on the graphical representation.
This popup contains a picture with a number block similar it is on your keyboard. Also you can change the sign.

Figure 2: Popup
 

Note: The TS_LetterKeys class is also working with Multi Language!
