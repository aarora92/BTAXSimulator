RS485 Status Popup

With a mouse click on an active RS485 card slot on the axs_pc104 panel you will get the profibus status popup.
 


In the RS485 status popup you can see the protocol name of your RS485 connection in the field SW, the status string  in the field Status and the running counter of you RS485 connection in the field RunCnt. You can see also the actual used channel in the field Actual Channel, the sum of input data in the field Input Data and the sum of output data in the field Output Data.
With the button Help you will get this online documentation of the RS485 setting and with the button Exit you can close the popup.
 
Status field color Description
Green Communication okay
Red (flashing) Communication not okay
Light Grey No Card

If your RS485 card doesn't run, your RS485 running counter will stop, and the staus field color will flash red.
