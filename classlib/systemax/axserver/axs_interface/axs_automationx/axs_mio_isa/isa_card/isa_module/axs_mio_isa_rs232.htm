RS232 Status Popup

With a mouse click on an active RS232 card slot on the MIO_ISA panel you will get the profibus status popup.
 


In the RS232 status popup you can see the protocol name of each channel of your RS232 connection in the fields SW CH0 - SW CH3, the status string  in the field Status and the running counter of you RS232 connection in the field RunCnt. You can see also the used channels in the field Channel, the sum of input data in the field Input Data and the sum of output data in the field Output Data.
With the button Help you will get this online documentation of the RS232 setting and with the button Exit you can close the popup.
 
Status field color Description
Green Communication okay
Red (flashing) Communication not okay
Light Grey No Card

If your RS232 card doesn't run, your RS232 running counter will stop, and the staus field color will flash red.
