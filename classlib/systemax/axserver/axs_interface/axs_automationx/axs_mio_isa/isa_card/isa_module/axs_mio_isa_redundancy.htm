Redundancy Status Popup

With a mouse click on an active redundancy card slot on the MIO_ISA panel you will get the Redundancy Status Popup.
 


In the field Redundancy you can see the name of the redundancy connection in the field RD-SW, the status in the field Com Ok and an redundancy error in the field Error Msg.
With the button Help you will get this online documentation of the Profibus setting and with the button Exit you can close the popup.
 
Status field color Description
Green Communication okay
Red (flashing) Communication not okay
Light Grey No Card

The Profibus settings of the redundancy are hard coded and cannot be changed by the user.
