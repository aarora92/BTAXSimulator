Graphical User Interface

CIF 60 PB Panel

The CIF 60 PB panel displays if there is an Hilscher PCMCIA Card existing and denotes the values of

  • active slaves
  • IO Points
  • Outputs and
  • Inputs


The LED display at the top of the panel indicates the error-state.
The following table provides a summary:
 

LED Color

Description

Green

There is no error.

Red

An error occurs.

Implementation Notes

The axs_CIF60_PB class is an integral part of the "axsystem" program. In terms of implementation it is required to have the "axsystem.tree" program installed into the running project. I.e. in the "Configuration" directory of the Project Window.
